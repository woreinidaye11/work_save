The goal is ideally to chose a model with the lowest test error. To achieve this we need to estimate the test error. This is done by two approaches:

  1. Indirectly estimate test error by adjusting the training error.
  2. Directly estimate test error using a validation set, or cross-validation.

Akaike information criterion(AIC ->smaller then better)

The AIC is defined as

AIC=2k−2ln(L^)AIC = 2k - 2 ln(\hat {L}) AIC=2k−2ln(L^)

where k is the number of parameters in the model, and ln(L^)ln(\hat{L})ln(L^) is the log likelihood evaluated at the maximum likelihood estimates. We choose the model with the lowest AIC.
*** In addition, the number of parameters in the model is the number of coefficients + the residual standard error. in the OQ5 will be 16+1 =17

Akaike information criterion corrected(AICc)

To adjust for small sample sizes, the AICc is used and it defined as

AICc=AIC+2k(k+1)n−k−1AICc = AIC +\frac{2k(k+1)}{n-k-1} AICc=AIC+n−k−12k(k+1)​

where n is the sample size.

Bayesian information criterion(BIC ->smaller then better))

A more stringent criterion with respect to the number of parameters is teh Bayesian information criterion(BIC). it is defined as

BIC=ln(n)k−2ln(L^)BIC = ln(n)k - 2ln(\hat{L}) BIC=ln(n)k−2ln(L^)

always pick the lowest one.

Bagging

Consider a problem with one very strong predictor, then all the trees will probably have this predictor as one of the early splits, and so all of the trees are similar and hence the predictions are highly correlated.

Random forest

Random forests decorrelate the trees.

  1. As bagging, we use bootstrapped training samples.
  2. Each time a split is considered, then a random sample of m predictors out of p possible predictors is chosen for the candidate split. A fresh sample of m predictors is chosen each tiem a split is considered.

Out-of-bag error estimation

On average each bagged tree uses about 2/3 of the observations. The remaining 1/3 of the observations not used are referred to as *out-of-bag(OOB) observations. So we can predict the value for an observation each time it is not used, and hence use this to give an overall prediction for the point

Variable importance

Random forests also have a OOB variable importance measure.

  1. For the bth tree, the OOB samples are predicted and the accuracy measured.
  2. The jth variables values for the OOB samples are permuted, and the accuracy is again calculated.
  3. The decrease in accuracy is averaged over all the trees and this is used as a measure for the jth variable.

Clustering

To find subgroups or clusters in the data.
To achieve this we need to define what it means for two objects to be similar or different

  1. PCA: finds a low dimensional representation of the data that explains most of the variance.
  2. Clustering: finds homogeneous subgroups of the data.

K-means clustering

Goal

Split the data into a pre-specified number of groups

Set-up

  1. Pre-specify the number of groups K
  2. Each observation is assigned to exactly one cluster

Notation

Good clustering

A good clustering is one that minimises the within-cluster variation, i.e., we want to solve

minC1,C2,…,Ck{∑k=1KW(CK)}min_{C_1,C_2,\dots,C_k} \{\sum_{k=1}^{K} W(C_K)\} minC1​,C2​,…,Ck​​{k=1∑K​W(CK​)}

Within-cluster variation - Squared Euclidean distance

Clustering

Algorithm

  1. Randomly assign a number of 1 to K to each observation
  2. Iterate unitl no changing in cluster assignment:
    a. For each cluster calculate the centroid. The centroid is the vector of the sample mean of the p features for observations in the cluster
    b. Assign each observation to the cluster of the centroid that is closest.

Cutting


Algorithm

  1. Start with n observations and (n2)\binom{n}{2}(2n​) pairwise dissimilarities.Treat each observations as its own cluster.
  2. For i=n,n−1,…,2:i =n,n-1,\dots,2:i=n,n−1,…,2:
    a. Fuse the two clusters of the i clusters that are most similar
    b. Compute the new pairwise inter-cluster dissimilarites between the remaining i-1 clusters.

Linkage

How do we compute dissimilarity between clusters?
This is done by calculating linkage:

  • Complete: largest dissimilarity between observation in first cluster and observations in second cluster.
  • Single: smallest dissimilarity between observation in first cluster and observations in second cluster.
  • Average: mean dissimilarity between observation in first cluster and observations in second cluster.
  • Centroid: distance between centroid of first cluster and centroid of second cluster.

Ward's method

Ward's minimum variance criterion minimizes the total within-cluster variance. To implement this method, at each step find the pair of clusters that leads to minimum increase in total within-cluster varinace after merging.
